Installing from Source
Clone the repository: git clone git://github.com/angular/angularjs-batarang
Navigate to chrome://chrome/extensions/ and enable Developer Mode.
Choose "Load unpacked extension"（加载正在开发的扩展程序）
Open the directory you just cloned (should open with Chrome, otherwise try dragging/dropping the file into Chrome) and follow the prompts to install.
